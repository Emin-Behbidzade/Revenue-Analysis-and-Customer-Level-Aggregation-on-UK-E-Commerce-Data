## Metod

* Dataset 541 909 satış əməliyyatı və 8 sütundan ibarətdir.
* Missing value analizi aparıldı.
* Description sütununda 1 454, CustomerID sütununda isə 135 080 boş dəyər aşkar edildi.
* InvoiceDate sütunu datetime formatına çevrildi və tarix əsaslı analiz üçün hazırlandı.
* Satışlar Country, Description və InvoiceDate üzrə qruplaşdırılaraq təhlil edildi.
* Satış həcmi və gəliri artırmaq üçün məhsul, müştəri və region səviyyəsində ilkin biznes nəticələri qiymətləndirildi.

## Əsas biznes nəticələri

* CustomerID məlumatı olmayan əməliyyatların payı yüksəkdir. Müştəri identifikasiyasının artırılması loyallıq proqramlarının qurulmasına, fərdiləşdirilmiş kampaniyalara və təkrar satışların artırılmasına imkan verə bilər.

* Description məlumatı olmayan məhsullar inventar və satış hesabatlarının dəqiqliyini azaldır. Məhsul məlumatlarının tamlaşdırılması ehtiyatların idarə edilməsini və satış analizlərini yaxşılaşdıracaq.

* Ölkələr üzrə satışların müqayisəsi göstərir ki, əsas gəlir bazarlarına daha çox marketinq və stok resursu ayrılması ümumi gəlirin artmasına müsbət təsir göstərə bilər.

* Tarix üzrə satış trendlərinin təhlili mövsümi dövrləri müəyyən etməyə imkan verir. Kampaniya və endirimlərin yüksək tələbat dövrlərinə uyğun planlaşdırılması satış həcmini artıra bilər.

* Yüksək satış həcmi yaradan məhsulların stok səviyyəsinin optimallaşdırılması və aşağı dövriyyəli məhsulların yenidən qiymətləndirilməsi əməliyyat xərclərini azaldaraq mənfəətin yüksəlməsinə kömək edə bilər.
